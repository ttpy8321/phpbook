키 목적
$_SERVER['REMOTE_ADDR'] 브라우저의 IP 주소
$_SERVER['HTTP_USER_AGENT'] 페이지를 요청하는 데 사용된 브라우저의 타입
$_SERVER['HTTP_REFERER'] 방문자가 링크를 통해 이 페이지에 온 경우라면, 브라우저는 이 페이지에 링크된 페이지의 URL을 보낼 수 있다. 모든 브라우저가 이 데이터를 보내는 것은 아니다.
$_SERVER['REQUEST_METHOD'] HTTP 요청 타입: GET 또는 POST
$_SERVER['HTTPS'] HTTPS를 사용하여 페이지에 접근한 경우에만 추가되며, 그 값은 true
$_SERVER['HTTP_HOST'] 호스트 이름(도메인 이름, IP 주소 또는 localhost일 수 있음)
$_SERVER['REQUEST_URI'] 이 페이지를 요청하는 데 사용된 호스트 이름 다음의 URI
$_SERVER['QUERY_STRING'] 쿼리 문자열의 모든 데이터
$_SERVER['SCRIPT_NAME'] 도큐먼트 루트 폴더에서 현재 실행 중인 파일까지의 경로
$_SERVER['SCRIPT_FILENAME'] 파일시스템 루트에서 현재 실행 중인 파일까지의 경로
$_SERVER['DOCUMENT_ROOT'] 파일시스템 루트에서 실행 중인 파일의 도큐먼트 루트까지의 경로
도큐먼트 루트와 파일세스템 루트의 차이점은 http://notes.re/php/filepaths를 참고하자.
