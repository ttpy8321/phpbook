대소문자 바꾸기

함수 설명
strtolower($string) 모든 문자가 소문자인 문자열을 반환한다.
strtoupper($string) 모든 문자가 대문자인 문자열을 반환한다.
ucwords($string) 모든 단어의 첫 글자가 대문자인 문자열을 반환한다.

문자와 단어 계수하기

함수 설명
strlen($string) 문자열의 문자 수를 반환한다.
공백과 구두점도 문자로 취급한다.
str_word_count($string) 문자열의 단어 수를 반환한다.

문자열에서 문자 찾기

함수 설명
strpos($string, $substring[, $offset]) 하위 문자열과 처음으로 일치하는 위치를 반환(대소문자 구분).
offset을 사용하면 그 문자 위치 다음부터 찾는다.
stripos($string, $substring[, $offset]) 대소문자를 구분하지 않는 strpos() 버전
strrpos($string, $substring[, $offset]) 하위 문자열에 대해 마지막으로 일치하는 위치를 반환(대소문자 구분)
strripos($string, $substring[, $offset]) 대소문자를 구분하지 않는 strrpos() 버전
strstr($string, $substring) 하위 문자열이 처음으로 일치하는 위치부터 문자열의 끝까지의 텍스트를 반환(대소문자 구분)
stristr($string, $substring) 대소문자를 구분하지 않는 strstr() 버전
substr($string, $offset[, $characters]) $offset에 지정된 위치로부터 문자열 끝까지의 문자를 반환. 만약에 $characters 매개변수가 사용되면 $offset 다음으로 반환할 수 있는 문자 수를 지정한다.
더 많은 옵션은 http://notes.re/php/substr을 참고하자.
* str_contains($string, $substring) 문자열에서 하위 문자열이 있는지 확인하여 true/false를 반환
