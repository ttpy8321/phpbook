레벨 설명
PARSE PHP 코드 구문의 오류는 PHP 인터프리터가 페이지를 전혀 실행하지 못하게 한다.
FATAL 실행 중 오류가 발생한 이후의 코드 실행을 중지하는 PHP 코드 오류
WARNING 문제의 원인이 될 수 있지만 페이지의 나머지 부분을 인터프리터가 실행하려고 시도한다.
NOTICE 문제를 지적할 수 있지만 페이지의 나머지 부분을 인터프리터가 실행하려고 시도한다.
DEPRECATED PHP의 향후 버전에서 제거될 수 있는 PHP 코드
STRICT 더 좋은 방법으로 작성할 수 있으며 향후 미래에 대비할 수 있는 PHP 코드
